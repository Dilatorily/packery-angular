paPackeryItem
directive in module packery-angular

Description

The pa-packery-item directive marks a HTML element to contain an item in a Packery instance.

Usage

as element:
<pa-packery-item>
</pa-packery-item>
as attribute
<ANY pa-packery-item>
   ...
</ANY>

Example

The following HTML code shows the usage of the pa-packery-item directive in the template.

<pa-packery>
    <pa-packery-item>...</pa-packery-item>
</pa-packery>
