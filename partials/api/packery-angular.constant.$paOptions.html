$paOptions
object in module packery-angular

Description

This service is an AngularJS constant that corresponds to the default values of the options to control both Draggabilly and Packery.

The Draggabilly default option is as follow:

  • handle: ''

The Packery default options are as follow:

  • itemSelector: '.pa-item'
  • columnWidth: 1
  • rowHeight: 1
  • stamp: '.pa-stamp'

Moreover, there are more default options to control the behavior of the Packery instances:

  • isAppended: true
  • isDraggable: true

Properties

    The default value is 1.

  • isDraggable
