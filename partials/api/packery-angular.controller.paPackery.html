paPackery
controller in module packery-angular

Description

The paPackery controller contains all the logic to manage a packery-angular instance. It wraps the Packery instance and exposes an API to add and remove Packery items.

This controller also wraps and extends the native Packery events. Those events are emitted them through the $rootScope.
